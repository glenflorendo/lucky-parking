# @lucky-parking/utilities

## Overview

The `@lucky-parking/utilities` package is a collection of common utility functions designed to enhance the overall functionality of our projects. These utilities cover a range of areas such as date formatting, enum handling, and string operations. By centralizing these functions in a dedicated package, we aim to promote code reusability and maintainability across our monorepo.

## Installation

To allow another package to install and consume the `@lucky-parking/utilities` package, run the following command:

```bash
pnpm add @lucky-parking/utilities
```
